# SQL-Analysis-on-Sales-and-Staff-Performance
This project is a collection of SQL solutions as data analyst  using joins, subqueries, self-joins, CTEs, and window functions. These exercises are based on a relational business database and demonstrate practical SQL techniques for data analysis and reporting.
## Topics Covered in this query
- Aggregate Functions (SUM, AVG, COUNT)
- Filtering and Grouping
- Multi-table Joins
- Self Joins & Hierarchical Data
- Correlated Subqueries
- Common Table Expressions (CTEs)
- Recursive CTEs
- Window Functions (RANK, Running Totals)
- Business Reporting Queries
